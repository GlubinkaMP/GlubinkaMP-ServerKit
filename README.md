# Glubinka MP — ServerKit

Готовый серверный комплект для запуска сервера Glubinka MP.

## Скачать
Открой раздел **Releases** справа на странице и скачай последнюю версию архива `GlubinkaMP-ServerKit`.

## Что внутри
- `GlubinkaMP.Server.exe`
- `settings.xml`
- `start.bat`
- папка `bin`
- папка `resources`

## Быстрый запуск
1. Скачай ZIP из раздела Releases
2. Распакуй архив в любую папку
3. Запусти `start.bat`

## Настройка
Открой файл `settings.xml` и измени нужные параметры:
- имя сервера
- порт
- лимит игроков
- список ресурсов

## Ресурсы
Свои ресурсы добавляй в папку `resources`.

## Важно
Это **готовый пакет**, а не исходный код ядра.
